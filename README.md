# SmartShop Customer Experience Optimization - Business Analysis Case Study
Microsoft Visio, Customer Journey Mapping, Stakeholder Analysis, Customer PainPoints

## Purpose 

This case study demonstrates core business analysis skills, including requirements analysis, process mapping, and customer experience evaluation.

This project is a business analysis case study focused on improving the customer journey and system processes for SmartShop, an online supermarket platform. The analysis explores why customers abandon carts, why delivery updates are inconsistent, and how data flows across key systems.
